DECLARACIÓN DE ACCESIBILIDAD

Alcance

Esta declaración describe el nivel de accesibilidad de las páginas públicas del proyecto INMOLINK. Incluye medidas técnicas y de contenido adoptadas para mejorar la accesibilidad.

Estado de conformidad

Se ha diseñado el sitio con criterios de accesibilidad web. Nuestro objetivo es cumplir, en lo posible, las WCAG 2.1 Nivel AA. No obstante, puede haber páginas o elementos que no alcancen todavía todos los requisitos.

Características de accesibilidad implementadas

  • Uso de HTML semántico (header, nav, main, article, footer, form).
  • Contrastes de color pensados para lectura.
  • Hojas de estilo alternativas disponibles: modo noche, alto contraste, texto grande y combinación.
  • Control de foco visible en enlaces y controles.
  • Diseño adaptable (responsive) y tamaños táctiles mínimos para botones.

Cómo activar los estilos accesibles

Hemos publicado varias hojas de estilo alternativas que puedes seleccionar desde el navegador. Los títulos de estilo disponibles son:

  • Estilo principal (por defecto)
  • modoNoche (tema oscuro)
  • altoContraste (alto contraste)
  • textoGrande (tamaño de letra aumentado)
  • textoGrandeDislexia (tamaño aumentado + OpenDyslexic)
  • altoContraste+textoGrande (combinado)

En Mozilla Firefox (recomendado para pruebas)

  1. Abre la página.
  2. Pulsa Alt para mostrar la barra de menú.
  3. Ve a Ver > Estilo de página y selecciona el estilo deseado.

Si usas Google Chrome

Chrome no ofrece menú nativo para seleccionar estilos alternativos. Para probar una hoja de impresión o emular la impresión, usa la vista previa de impresión o las herramientas de desarrollador (Emulate CSS media → print). Si deseas que los usuarios cambien estilos desde la propia web, contacta con nosotros (más abajo hay un formulario) y te indicamos una solución basada en JavaScript y cookies.

Tipografía y lectura

En estilos accesibles hemos incluido la opción de usar OpenDyslexic.

Navegación y teclado

La navegación por teclado está soportada: puedes recorrer enlaces y controles con Tab. Los elementos interactivos muestran un foco visible.

Cómo reportar incidencias o pedir alternativas

Si encuentras problemas de accesibilidad, por favor, contáctanos usando el formulario debajo o envía un correo a daw@inmolink.example. Incluye:

  • URL donde has encontrado el problema.
  • Descripción del problema y dispositivo/navegador usado.
  • Tu propuesta (si tienes) o captura de pantalla.
